## 1. 日勤クリニックのシフト構造

### 1.1 典型的な診療時間帯と勤務パターン

日本の一般的な外来クリニック（診療所）は、夜勤を持たない「日勤のみ」で運営される。以下がよく見られる勤務パターンである。

#### 標準的な診療時間帯

| 診療区分 | 診療時間例 | 備考 |
|---------|-----------|------|
| 午前診療 | 09:00〜12:30 | 内科・小児科等で多い |
| 午後診療 | 14:00〜18:00 または 15:00〜18:30 | 診療科により異なる |
| 全日診療 | 09:00〜18:00（昼休憩あり） | 皮膚科・整形外科等で多い |
| 半日診療（週次） | 土曜午前のみ等 | 週ごとの変動あり |

#### 典型的なシフト区分

| シフト区分 | 勤務時間例 | 労働時間 | 主な対象者 |
|-----------|-----------|---------|-----------|
| 全日（フル） | 08:30〜17:30（休憩60分） | 8時間 | 常勤スタッフ |
| 午前（AM） | 08:30〜13:30 | 4.5〜5時間 | パート・時短 |
| 午後（PM） | 13:30〜18:30 | 4.5〜5時間 | パート・時短 |
| 早番 | 08:00〜16:00 | 7.5時間 | 準備担当者 |
| 遅番 | 10:00〜18:30 | 8時間 | 後片付け担当者 |

> **日勤クリニック特有**: 夜勤明けや深夜帯の引き継ぎが不要なため、シフト設計は比較的単純だが、「午前診療終了〜午後診療開始」の昼休憩時間帯の配置調整が重要な要素となる。

### 1.2 診療科ごとの必要人員・役割

クリニックで勤務するスタッフの役割と必要な配置を以下に整理する。

| 役割 | 必要資格 | 配置の考え方 |
|------|---------|------------|
| 医師 | 医師免許（専門医資格は診療科による） | 診療時間中は最低1名必須。複数医師クリニックは診療科別に配置 |
| 看護師 | 看護師免許 | 診療補助・処置担当。医師1名につき1〜2名が目安 |
| 准看護師 | 准看護師免許 | 看護師の指示のもとで業務。看護師との比率を要管理 |
| 医療事務 | 資格任意（調剤補助は別） | 受付・会計・レセプト業務。診療時間中1名以上必須 |
| 受付補助 | 不要 | ピーク時の補助。パート・時短対応多い |
| 調剤薬剤師 | 薬剤師免許 | 院内処方のあるクリニックのみ |
| PT/OT/ST | 各免許 | リハビリ科があるクリニックのみ |

### 1.3 半日勤務・パート勤務の取り扱い

日勤クリニックではパート・時短勤務者が多く、以下の管理が必要：

- **週20時間以上のパート**: 社会保険・雇用保険の適用対象（2022年改正）
- **週10〜20時間未満のパート**: 午前のみ・午後のみの固定シフトが多い
- **育児短時間勤務者**: 法定6時間勤務（所定8時間からの短縮）
- **曜日固定パート**: 特定曜日のみ出勤。シフト設計時に固定枠として扱う

### 1.4 休憩時間の制約

| 勤務時間 | 法定休憩時間 | クリニックでの実態 |
|---------|------------|-----------------|
| 6時間以下 | 休憩不要（法律上） | 午前・午後パートは休憩なしも可能 |
| 6時間超〜8時間以下 | 45分以上 | 昼休み（診療休止時間）に充当することが多い |
| 8時間超 | 60分以上 | 全日勤務者は昼休みを1時間確保 |

> **注意**: 診療昼休憩中であっても「待機状態（呼び出し可）」であれば労働時間とみなされる可能性がある。スタッフが自由に使える昼休みとの区別が重要。

## 2. ハード制約（絶対遵守）

ハード制約は、違反した場合に「シフト不成立」または法律違反・医療事故リスクが生じる制約である。

### 2.1 最低配置人員（診療時間帯別）

| 診療時間帯 | 最低配置 | 根拠 |
|-----------|---------|------|
| 診療中（午前） | 医師1名以上 + 看護師（または准看護師）1名以上 + 受付1名以上 | 医療法・診療所開設要件 |
| 診療中（午後） | 同上 | 同上 |
| 診療外（開業準備・後片付け） | スタッフ最低1名 | 施設管理上の要件 |
| 土曜午前のみ診療 | 医師1名以上 + 最低限のサポートスタッフ | 診療所開設要件 |

> **クリニック特有**: 病院とは異なり、外来のみの診療所（クリニック）は入院患者がいないため、「患者数に比例した入院看護師配置基準（医療法21条）」は直接適用されない。ただし、各診療科の医師は必ず診療時間中に在院しなければならない。

### 2.2 連続勤務日数上限

| 区分 | 上限 | 備考 |
|------|------|------|
| 連続勤務日数 | 6日以内（推奨：5日以内） | 労働基準法上の4週4日休日確保から逆算 |
| 週内連続フルシフト | 5日以内（週休2日制の場合） | 週40時間超となるため |

> **日勤クリニックでの実態**: 土曜が半日診療のため、「月〜金フル + 土曜午前」という週6日勤務パターンに注意が必要。この場合でも連続6日以内を守る必要がある。

### 2.3 公休日数の確保（週・月）

| 単位 | 最低休日日数 | 法的根拠 |
|------|------------|---------|
| 週次 | 1日以上 | 労働基準法第35条 |
| 4週単位 | 4日以上 | 変形休日制採用時 |
| 月次（4週5日制の場合） | 8日以上が目安 | 週休2日制の慣行 |

### 2.4 スキル・資格要件

| チェック項目 | 内容 |
|------------|------|
| 医師在院確認 | 診療時間中、当該診療科の医師免許保有者が必ず院内にいること |
| 看護師免許確認 | 医師の指示なしに実施できる業務（採血・処置等）には看護師免許が必要 |
| 特定行為看護師 | 高度な医療行為を行う場合は特定行為研修修了者が必要 |
| 調剤薬剤師 | 院内処方（院内調剤）実施時は薬剤師が必須 |
| 同一役職の重複禁止 | 院長（管理医師）は1名のみ。複数診療科を持つ場合は科長を別途指定 |

### 2.5 勤務間インターバル（日勤のみ）

| 制約 | 内容 |
|------|------|
| 前日の終業〜翌日の始業 | 11時間以上（厚生労働省推奨・努力義務） |
| 遅番（18:30終業）→翌日早番（08:00始業） | インターバル13.5時間 → 問題なし |
| 残業延長時 | 診療延長で20:00終業→翌日08:30始業だと8.5時間 → 努力義務違反リスク |

> **日勤クリニック特有**: 夜勤がないため深刻なインターバル問題は少ないが、診療延長による残業が続いた場合に翌日始業が早いスタッフへの影響を管理する必要がある。

## 3. ソフト制約（できるだけ遵守）

ソフト制約は、違反すると品質（スタッフ満足度・公平性）が低下するが、シフト自体は成立する制約である。違反には重み付きペナルティを設定して最適化する。

### 3.1 スタッフの希望休

| 項目 | 内容 | 優先度 |
|------|------|--------|
| 希望休（公休日指定） | 月○日分の希望日を反映 | 高（患者・家族行事等） |
| 希望勤務（特定日出勤希望） | 特定の曜日・時間帯の出勤希望 | 中 |
| 連休希望 | 複数日連続で休みたい希望 | 中 |
| 特定シフト回避 | 「月曜遅番は避けたい」等の個人事情 | 低 |

### 3.2 公平な勤務分配

| 公平性の観点 | 内容 |
|------------|------|
| 総勤務日数の均等化 | 常勤スタッフ間で月間勤務日数の差を最小化 |
| シフト区分の均等化 | 「遅番」「土曜出勤」等の不人気シフトが特定者に集中しないよう分散 |
| 土曜・祝前日出勤の均等化 | 土曜午前診療への出勤がスタッフ間で公平に分配 |
| 有給消化の促進 | 月次で有給残日数が多いスタッフを休みに割り当てやすくする |

### 3.3 連続勤務回数の平準化

| 制約 | 内容 |
|------|------|
| 4日以上連続勤務の最小化 | 連続4日以上の勤務が発生する場合はペナルティ |
| 連続2日以上休日の促進 | 2連休が取りやすいシフト設計 |
| 週に1回以上の休みパターン | 土日どちらか1日は必ず休日になるよう設計 |

### 3.4 特定ペア・チームの一緒配置希望

| 項目 | 内容 |
|------|------|
| チームの継続性 | 担当患者が多い診療日は特定の医師・看護師ペアで対応する |
| 新人・指導員の同時配置 | 新人スタッフは指導担当者と同じシフトに配置する |
| 特定の組み合わせ回避 | 相性・コミュニケーション上の問題があるペアの同時配置を避ける |
| 診療科専従スタッフ | 特定の診療科のみ対応可能なスタッフは、その診療科がある日に配置 |

### 3.5 パート・時短勤務者への配慮

| 制約 | 内容 |
|------|------|
| 固定シフトパートの希望尊重 | 「火・木午前のみ」等の固定スケジュールを優先 |
| 育児時短勤務者の制約 | 保育園のお迎え時間（16:00等）を超える勤務回避 |
| 週次勤務時間の上限管理 | パートの雇用契約時間（例：週20時間）を超えないよう管理 |

## 4. 日本の法律・規制（日勤のみ適用）

### 4.1 労働基準法（日勤クリニックへの適用）

| 条項 | 内容 | シフトへの影響 |
|------|------|--------------|
| 法定労働時間（第32条） | 1日8時間・週40時間 | これを超えた分は時間外労働として割増賃金が必要 |
| 法定休日（第35条） | 毎週1日または4週4日以上 | 週1日以上の休日をシフトで確保する |
| 休憩時間（第34条） | 6時間超→45分、8時間超→60分 | 全日勤務者は昼休みを60分確保する |
| 時間外労働上限（第36条） | 月45時間・年360時間（特別条項：年720時間） | 残業が多くなる月は翌月に勤務を軽減して調整 |

> **深夜割増（第37条）・夜勤規制は対象外**: 日勤のみのクリニックでは22:00〜05:00の深夜帯は発生しないため、深夜割増賃金および夜勤関連規制（夜勤回数上限・夜勤明け翌日休み等）は適用されない。

### 4.2 休憩時間の詳細（日勤クリニック）

```
【6時間以下の勤務（午前・午後パート）】
  → 法定休憩は不要
  → ただし、診療の合間に適宜短休憩を取らせることが望ましい

【6時間超〜8時間以下の勤務（早番・遅番）】
  → 45分以上の休憩を確保
  → 診療の昼休み（12:00〜14:00等）に充当することが多い

【8時間超の勤務（全日フル勤務）】
  → 60分以上の休憩を確保
  → 昼休みの診療停止時間（約1〜2時間）で対応

【休憩の一斉付与の原則（第34条2項）】
  → 一斉付与が原則だが、医療業は例外（シフトごとに個別に取得可）
```

### 4.3 週40時間規制と変形労働時間制

クリニックでは診療曜日・時間が週によって異なる場合があり、変形労働時間制の活用が有効。

| 制度 | 内容 | クリニックでの活用例 |
|------|------|------------------|
| 1ヶ月単位の変形労働時間制 | 1ヶ月の総労働時間内で日・週の上限を超えることが可能 | 土曜出勤月はその週の平日勤務を短縮して調整 |
| フレックスタイム制 | コアタイム設定で出退勤時間を柔軟化 | 医療事務・事務系スタッフへの適用が多い |

### 4.4 月次残業時間の上限（36協定）

```
通常の36協定:
  月45時間以内・年360時間以内

特別条項付き36協定（繁忙時）:
  年720時間以内（月100時間未満・連続2〜6ヶ月平均80時間未満）

シフト設計の目標:
  各スタッフの月次残業を原則20時間以内に抑えるシフト設計
  （45時間上限の半分程度を目標とすることで余裕を持たせる）
```

### 4.5 有給休暇管理（年5日取得義務）

| 項目 | 内容 | シフトへの実装 |
|------|------|--------------|
| 付与条件 | 年10日以上の有給休暇が付与された労働者 | 常勤・週4日以上勤務のパートが対象 |
| 取得義務 | 年5日の有給取得が使用者の義務 | シフト作成時に有給消化残日数を参照し、自動的に有給シフトを配置 |
| 時季指定義務 | 年度末（3月）までに5日に満たない場合、使用者が日を指定 | 1〜3月のシフト作成時に有給消化が遅れているスタッフを優先的に休みに配置 |

### 4.6 医療法上の人員配置基準（クリニックへの適用）

| 種別 | 内容 |
|------|------|
| 診療所（クリニック）の開設要件 | 医師1名以上（開設者が医師または医療法人等）。外来のみの場合、入院患者比例の看護師配置基準は適用されない |
| 特定診療科の追加要件 | 産婦人科・放射線科等では追加の有資格者配置が必要な場合がある（医療法施行規則） |
| 管理医師の在院義務 | 診療時間中、管理医師（または代理医師）が在院していること |
| 院内感染対策 | 感染対策担当者（医師または看護師）の指定が必要（医療法第6条の12） |

## 5. 評価指標

シフトの品質を客観的に測定するための指標を以下に整理する。

### 5.1 ハード制約充足率（最優先）

```
ハード制約充足率 = 充足された制約数 / 全ハード制約数 × 100%

目標: 100%（1件でも違反があれば「シフト不成立」）

主なチェック項目:
  - 診療時間帯における医師在院確認: ✅/❌
  - 診療時間帯における最低看護師数確認: ✅/❌
  - 法定週休の確保: ✅/❌
  - 休憩時間の確保: ✅/❌
  - 連続勤務上限の遵守: ✅/❌
```

### 5.2 希望充足率

```
希望充足率 = 反映された希望日数 / 申請された希望日数 × 100%

目標: 80%以上（全希望を満たすことは困難なため現実的な目標値を設定）

内訳:
  - 希望休充足率: 公休日指定希望のうち実際に反映された割合
  - 希望シフト充足率: 特定シフト（AM/PM等）希望の反映率
  - 連休希望充足率: 連続休暇希望の充足率
```

### 5.3 公平性指標

| 指標 | 計算方法 | 目標値 |
|------|---------|--------|
| 月間勤務日数の標準偏差 | 常勤スタッフ間の月次勤務日数のσ | σ ≤ 1.0日 |
| 土曜出勤回数の差分 | max(土曜出勤数) - min(土曜出勤数) | 差 ≤ 1回/月 |
| 遅番・不人気シフト回数の標準偏差 | スタッフ間のσ | σ ≤ 1.0回/月 |
| 有給取得率 | 取得日数 / 付与日数 × 100% | 年5日以上（法的義務）|

### 5.4 残業最小化

```
月間残業時間 = Σ(各スタッフの月間残業時間)

目標: 全スタッフの月間残業時間 ≤ 20時間
警告ライン: 月間残業時間 > 45時間（36協定上限）

診療延長リスク対策:
  - 診療終了予定時間より30分の余裕を持ったシフト設計
  - 遅番スタッフは診療終了後の後片付け時間も考慮
```

### 5.5 スタッフ満足度（オプション）

定量化は難しいが、以下の代理指標でモニタリング可能：

| 代理指標 | 計算方法 |
|---------|---------|
| 希望充足率（再掲） | 希望反映率を高めることで満足度向上 |
| 公休の連続性 | 2連休以上の確保率（単日休みより連休を好む傾向） |
| 連続勤務日数の平均 | 短いほどスタッフへの負担が少ない |
| 不人気シフトの偏り係数 | ジニ係数 ≤ 0.2を目標 |

## 6. 日勤クリニック特有の設計ポイント（まとめ）

### 夜勤不要による設計の単純化

```
✅ 除外できる要素:
  - 夜勤回数の管理・均等化
  - 夜勤明け翌日の休日確保
  - 深夜割増賃金の計算
  - 夜勤後の勤務間インターバル管理
  - 夜勤専従スタッフの特別管理
  - 2交代/3交代の切り替え管理

⚠️ 日勤特有の注意点:
  - 「土曜午前診療」への出勤管理（土曜は半日でも必ず出勤が必要）
  - 「昼休み時間帯の配置」（昼休みを完全に休憩にできるかの確認）
  - 「パート・時短勤務者の比率が高い」ことによる人員確保の難しさ
  - 「診療延長残業」の管理（夜まで延びることはなくても、日中の延長は頻繁）
```

### 診療カレンダーとの連動

```
クリニック特有の勤務日マスタ:
  - 診療日（平日）: 月〜金、全日または午前・午後
  - 診療日（土曜）: 午前のみが多い（または全休）
  - 休診日: 日曜・祝日・盆休み・年末年始・院長学会出張日等
  - 臨時休診: 感染症等による突発的な休診

シフト設計時は「診療カレンダー」に基づき、
  - 休診日は全スタッフ休み（または最小限の事務スタッフのみ）
  - 土曜半日診療日は出勤者を絞った配置
```
